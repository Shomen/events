# Events (Nuxt + Vue)

A responsive event booking app where users can filter events by **category** and **month**, an event page for details and further booking, and a **dummy** multi-step booking flow.

## Live Demo
Check it out here: [Live demo](https://events-woad-seven.vercel.app/)

## Features
- Home landing page with curated copy and featured event highlight
- Events list with category filter + month filter
- Event detail with a 4-step booking UI (dummy flow)
- Contact page with a styled form shell and demo submit state
- About page with dummy content (same visual style)

## Data
- Event catalog stored in json file

## Technologies
- Nuxt 4 + Vue 3 + TypeScript
- Tailwind CSS 
- Styling driven by a custom theme (colors, fonts, backgrounds)
